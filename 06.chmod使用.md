#### 权限管理
  * ###### r：读
  * ###### w：写
  * ###### x：执行

```shell
chown：改变文件属主（只有管理员可以使用次命令）
chown xxx：xxx  file 
chown --reference=/xx/xxx
```

<br>

#### chgrp 修改属组依然只有管理员才可以使用此命令
  * ###### -R
  * ###### --reference=/xxx/xxx      

<br>

#### chown ：GRPNAME  file....

#### chgrp USERNAME：GRPNAME file....

chmod: 修改文件的权限修改三类用户的权限
```shell
chmod 775 file... -R
```
