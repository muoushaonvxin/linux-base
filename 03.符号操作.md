#### 此处文档
```shell
cat << END
>dafjdakf
>dajflkdaf
>dafjdakl
>END
```


管道：     前一个命令的输出，作为后一个命令的输入


命令1     |       命令2         |   命令3  | .....



#### .. ：回到上一级目录当中


注意：
删除一个文件一般用 rm file ...　次法会提示你确认
推荐初学使用，rm -rf file 表示不提示直接删除

对于文件来说rm -rf 工具前置删除目录不提示，非常危险，很多高手都出现过严重的错误。
