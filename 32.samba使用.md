认证模式
* share最低级别所有人可以访问
* user基于用户认证用户密码
* server需要一台专门的用户密码认证的server
* domain AD域作为AD域成员机器
* ADS AD活动目录有了AD域的管理员

认证模式
```shell
security = share
```

```shell
[root@zhangyz ~]# mkdir /share

writable = yes # 可写如果=no表示只读

printable = yes # 指定为打印机
read_only = yes # 只读
```
